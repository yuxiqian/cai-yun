# 彩云

> Dedicated to **her**.

## 介绍

 * 展示数百种中国 & 日本传统色彩。
 
 * 使用 RGB、HSL、CMYK 和十六进制数等多种方式表示色彩。
 
 * 仔细调整的界面动画与 macOS "Mojave" 暗 / 亮模式支持。

 * 希望你喜欢。
